# Unity3D-Gameplay-Demo

Prototype third-person Unity project focused on core player movement and camera systems.

## Current Features
- Player movement (walk, jump)
- Directional facing logic
- Basic third-person camera follow
- Camera rotation via input

## Project Status
Early prototype. Focus is currently on core controls and feel before adding gameplay systems.

## Tech
- Unity
- C#
- Git / GitHub

## Notes
-Assets used for presentation; all gameplay systems implemented by me.
-Built using Unity documentation and standard engine patterns.
-Camera implemented using standard third-person Unity patterns, adapted and modified for this project.
